# Moonify logger

### Grabify-like alternative, sends the same info Grabify sends you, but instead it sends it to your discord web-hook.

### Setup

1. Download the code
2. Open script.js with any text editor
3. Change the webhook
4. Save the file
5. Open index.html
6. Check your discord webhook.

### Features

- Browser info
  - Operating System
  - IP Address
  - Country
  - Date/Time
  - Browser
  - User Agent
  - Referring URL
  - Host Name
  - ISP
